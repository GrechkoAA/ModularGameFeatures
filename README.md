# Modular Game Features (Модульные игровые функции)
Набор многократно используемых, готовых к внедрению в производство функций Unity, разработанных как независимые модули.
Каждая функция изолирована, расширяема и легко интегрируется в другие проекты.
Все модули покрыты юнит-тестами и легко проверяемы, что обеспечивает надежность и предсказуемость их работы.

> ⚡ **Каждый модуль сопровождается примером использования**, который демонстрирует основные возможности и способы интеграции в проект.

**Каждый модуль:**

- Является автономным
- Имеет минимальное количество зависимостей
- Следует принципам чистой архитектуры
- Разработан с учетом масштабируемости и расширения
- Может быть интегрирован в другой проект с минимальной настройкой

## 🧩 Доступные модули

### 💰 Currency Bank

Модуль управления игровыми валютами с изолированной бизнес-логикой и операциями над балансом.

**Возможности:**

- Хранение нескольких типов валют  
- Безопасные операции добавления, списания и установки баланса
- Проверка достаточности средств  
- Расширяемая архитектура (поддержка новых типов валют)  
- ✅ Полное покрытие юнит-тестами
